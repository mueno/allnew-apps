よくある質問
使い方や機能についてお答えします
どんな体重計に対応していますか？ +

デジタル表示の体重計に対応しています。

  • 対応: 7セグメントLED、液晶ディスプレイなどのデジタル表示
  • 非対応: アナログ体重計（針式メーター、回転式ダイヤル）
認識できる体重の範囲は？ +

30.0kg〜200.0kgの範囲に対応しています。この範囲外の数値は認識されません。

対応しているiPhoneは？ +

iOS 17.0以降を搭載し、カメラ機能およびApple HealthKitに対応したiPhoneで動作します。

無料で使えますか？ +

15回まで無料でお使いいただけます。16回目以降は一回限りの買い切り購入が必要です。

サブスクリプション（月額・年額課金）はありません。一度購入すれば、ずっとお使いいただけます。

データはどこに保存されますか？ +

体重データはお使いのデバイス内のApple HealthKit（ヘルスケア）に保存されます。

当社が運用するサーバーへの送信はありません。撮影した画像もOCR処理後に即時破棄されます。HealthKitの保存・同期（iCloud等）の挙動は、ユーザーの端末設定およびAppleの仕様に従います。

認識精度が低い場合はどうなりますか？ +

アプリが認識結果の精度（信頼度）を自動判定します。

  • 高信頼度の場合: 確認画面なしで自動的にヘルスケアに記録されます
  • 低信頼度の場合: 認識した値を確認画面で表示し、「この値で記録する」を選択した場合のみ保存されます。「記録しない」を選ぶと、再度撮影・認識を行います。

低信頼度と判定された場合、ユーザーが明示的に許可しない限り記録されることはありません。

認識結果が間違っていた場合は？ +

保存前の場合: 認識精度が低い場合は、アプリが確認画面を表示します。正しくないと思ったら「記録しない」を選択し、再度撮影してください。

保存後の場合: 高信頼度の認識結果は確認なしで自動保存されます。誤った数値が保存された場合は、iOSのヘルスケアアプリから削除してください。

削除手順:

  • ヘルスケアアプリを開く
  • 「ブラウズ」→「体の測定値」→「体重」を選択
  • 該当するデータを左にスワイプして「削除」

※HealthKitの仕様上、保存されたデータの編集はできません。削除して、本アプリで再認識・再登録する必要があります。

アプリを削除するとデータは消えますか？ +

アプリを削除する際、iOSが「ヘルスケアデータを削除しますか？」と確認します。

  • 「削除」を選択 → ヘルスケアのデータも削除されます
  • 「削除しない」を選択 → データはヘルスケアアプリに保持されます
購入を復元したい +

過去に購入された方は、再購入の必要はありません。

復元手順: アプリを開き「設定」→「アプリについて」→「購入を復元」をタップしてください。

認識精度を上げるコツは？ +
  • 体重計のディスプレイが明るく表示されている状態で撮影
  • ガイド枠内に数字が収まるように位置を調整
  • 手ブレを避け、できるだけ静止した状態で撮影
  • ディスプレイの汚れや劣化がないか確認
  • 直射日光や強い照明の反射を避ける

プライバシーポリシー
あなたのデータを大切に扱います
最終更新: 2026年2月4日

重要: 本アプリは、ユーザーの体重データを当社の外部サーバーに送信・保存しません。すべてのデータはお使いのデバイス内およびApple HealthKitに保存されます。HealthKitの保存・同期の挙動（iCloud等）は、ユーザーの端末設定およびAppleの仕様に従います。

1. 収集する情報の種類

  • カメラで取得する画像: OCR処理後に即時破棄。当社が運用するサーバーへの送信はありません。
  • 健康関連データ（HealthKit）: 体重データの書き込み・読み取り。お使いの端末およびApple HealthKitに保存されます。HealthKitの保存・同期の挙動（iCloud等）は、ユーザーの端末設定およびAppleの仕様に従います。
  • デバイス情報: iPhone機種識別（重量差し引き機能用）。端末内で処理され、当社が運用するサーバーへの送信はありません。
  • デバイス内保存情報: 無料使用回数、購入状態、設定情報。当社が運用するサーバーへの送信はありません。
  • 収集しない情報: 本アプリでは広告SDKやアナリティクスサービスを使用せず、ユーザー登録機能もありません。

2. 情報の利用目的

体重の記録と健康管理支援、体重変化の表示、ウェルネスメッセージの表示、iPhone重量の自動差し引き。これ以外の目的には利用しません。

3. 第三者への提供

当社は画像データやHealthKitデータを第三者に提供、販売、共有しません。

4. Apple HealthKit特記事項

  • HealthKitデータを広告、マーケティング、データマイニング目的で使用しません
  • HealthKitデータをデータブローカーに販売しません
  • 当社はHealthKitデータを当社の外部サーバーに送信・保存しません。HealthKitの保存・同期（iCloud等）の挙動は、ユーザーの端末設定およびAppleの仕様に従います
  • HealthKitへのアクセスはiOSの設定からいつでも取り消せます

5. Apple Intelligence（オンデバイスAI処理）

本アプリはApple Intelligenceを使用してウェルネスメッセージを生成する場合があります。処理は原則として端末内で行われますが、Appleの仕様によりPrivate Cloud Compute等で処理される場合があります。当社は、当該処理のために独自の外部サーバーへデータを送信・保存しません。

6. セキュリティ対策

データ最小化設計、可能な限り端末内での処理、Keychainによる機密情報保護、HealthKitの暗号化による保護を採用しています。Apple Intelligenceを利用する処理は、Appleの仕様によりPrivate Cloud Compute等で処理される場合があります。

7. 子供のプライバシー

本アプリは13歳未満のお子様を対象としていません。アプリの機能利用において、当社は個人情報を収集しません。

8. ユーザーの権利

  • すべてのユーザー: HealthKitデータの管理・削除は「設定」>「ヘルスケア」から可能。アプリ削除時、iOSがヘルスケアデータの削除を確認します。
  • EU/EEA居住者（GDPR）: 苦情申立て権、同意の撤回権。
  • カリフォルニア州居住者（CCPA/CPRA）: 知る権利、削除権、オプトアウト権。
  • 中国居住者（PIPL）: 知る権利、閲覧・複製権、削除権。当社はユーザーデータを当社の外部サーバーへ移転しません。ただし、HealthKitの同期（iCloud等）やApple IntelligenceのPrivate Cloud Compute等、Appleのサービスにより、ユーザーの国・地域外で処理／保存される可能性があります。
  • 韓国居住者（PIPA）: 処理中止要求権。
  • ブラジル居住者（LGPD）: データ移行要求権。

9. お問い合わせについて

ユーザーが当社へお問い合わせを行う場合、当社は返信対応のために、メールアドレスおよびお問い合わせ内容を取得・利用します。これらの情報はサポート対応の目的にのみ用い、法令に基づく場合を除き第三者に提供しません。対応完了後は合理的な期間で削除または匿名化します。

10. 準拠法と管轄

本ポリシーは日本法を準拠法とします。各地域の居住者には該当する法律（GDPR、CCPA/CPRA、PIPL、LGPD、PIPA）が適用されます。

3. 本アプリの概要と対応機器

本アプリは、体重計のディスプレイに表示された数値をカメラで撮影し、AI（人工知能）による光学文字認識（OCR）技術を用いて読み取り、Apple HealthKit（ヘルスケア）に記録するアプリケーションです。

  • 対応体重計: デジタル表示（7セグメントLED、液晶ディスプレイ等）の体重計に対応しています。アナログ体重計（針式メーター、回転式ダイヤル等）には対応しておりません。
  • 対応体重範囲: 30.0kg〜200.0kgの範囲に対応しています。
  • 対応デバイス: iOS 17.0以降を搭載したiPhone。

  • HealthKitデータをデータブローカーや第三者に販売・提供しません

7. OCR認識の精度について

  • 認識精度の非保証: OCR認識の精度は100%ではありません。
  • 高信頼度結果の自動保存: 本アプリが認識結果を高信頼度と判定した場合、ユーザーの確認なしに自動でApple HealthKit（ヘルスケア）へ記録します。
  • 低信頼度結果の確認フロー: 本アプリが認識結果を低信頼度（精度が十分でない）と判定した場合、認識した値を画面に表示し、ユーザーが内容を確認した上で「この値で記録する」ことを許可した場合にのみ、ヘルスケアへ記録します。ユーザーが許可しない場合、その値は記録されず、再度撮影・再認識を行っていただきます。
  • 記録後の訂正方法: Apple HealthKitに記録された体重データは、本アプリから編集できません（削除のみ可能）。誤った値を修正したい場合は、iOSの「ヘルスケア」アプリで該当データを削除した上で、本アプリで再度認識・記録を行ってください。
